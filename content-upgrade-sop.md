# 钱智汇 · 文章翻新 SOP v2.0

## 质量标准卡

| 维度 | 旧标准 | 新标准 |
|------|--------|--------|
| 正文字数 | 700-1200字 | **≥2000字**（不含FAQ/免责声明） |
| 开篇钩子 | 概念定义 | **真实场景/数据冲击/痛点提问** |
| 数据引用 | 0-1个无出处 | **≥2个有来源标注**（银保监/央行/行业协会） |
| 案例深度 | 1句话压缩 | **≥1个完整案例（≥300字，含人物+背景+决策+结果+反思）** |
| 对比表格 | 偶尔有 | **≥1个结构化对比表** |
| 实操步骤 | 泛泛而谈 | **≥3个可执行步骤（含具体数字/工具/时间节点）** |
| 常见误区 | 无 | **≥2个避坑指南（反面教材+正确做法）** |
| FAQ | 1-2句百科级 | **每问≥100字，有场景+数据+建议** |
| 配图 | 无 | **2-3张（开篇场景图+数据图表+流程图）** |
| 语气 | 教科书式 | **自媒体亲和力：用"你"、用类比、用短句** |

## 新文章结构

```
1. 封面图 (ImageGen: 1200x630 场景插图)
2. 开篇钩子 (150-200字)
   - 一个真实场景 或 一个冲击数据 或 一个痛点问题
3. 核心拆解 (300-400字)
   - 用类比解释复杂概念
   - 搭配信息图/对比表
4. 实操方法论 (500-800字)
   - ≥3步操作流程
   - 每步含具体数字/工具/时间
   - 搭配流程图/步骤图
5. 深度案例 (300-500字)
   - 真实人物 + 背景 + 决策过程 + 财务结果 + 反思
   - 搭配数据图表
6. 避坑指南 (200-300字)
   - ≥2个常见误区（反面教材 + 正确做法）
   - 对比表格
7. 一句话总结 (50-80字)
8. FAQ (3问，每问≥100字)
9. 免责声明
```

## 配图规范

| 图序 | 位置 | 类型 | 尺寸 | 内容 |
|------|------|------|------|------|
| 图1 | 标题下方 | 场景插图 | 1200x630 | 主题相关的金融生活场景 |
| 图2 | 实操方法论中 | 信息图/流程图 | 800x600 | 步骤图或概念对比 |
| 图3 | 深度案例旁 | 数据图表 | 800x500 | 收益对比/流程图 |

图片要求：
- `footnote` 设为 ""（无AI水印）或"钱智汇"
- `style` 统一用 "professional illustration, clean, modern, financial"
- `quality` 设为 "high"
- 生成到 `images/` 目录，文章中引用相对路径

## 翻新流程

1. 读取原文
2. 按新结构重写（保留核心信息，补数据/案例/误区/对比表）
3. 生成2-3张配图（ImageGen）
4. 更新 frontmatter（添加 image 字段指向封面图）
5. 更新 SEO 标签
6. 验证字数≥2000 + 数据引用≥2 + 案例≥300字
