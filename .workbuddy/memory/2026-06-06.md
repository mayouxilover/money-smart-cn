## Google Consent Mode v2 + CMP GDPR 合规（2026-06-06 11:59）

### 背景
AdSense 审核状态变为「正在准备」，用户通过 Google 内置 CMP 创建了含"同意"和"管理"选项的 GDPR 消息。需要配合 Consent Mode v2 代码。

### 实现
- 在 `convert_md_to_html.py` 所有 8 个 `<head>` 模板中，AdSense 脚本之前插入 Consent Mode v2 默认代码：
  ```javascript
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('consent', 'default', {
    'ad_storage': 'denied',
    'ad_user_data': 'denied',
    'ad_personalization': 'denied',
    'analytics_storage': 'denied',
    'wait_for_update': 500
  });
  ```
- f-string 转义处理：`{` → `{{`，`}` → `}}`
- Google CMP 横幅在用户选择后自动调用 `gtag('consent', 'update', {...})`

### 技术点
- `wait_for_update: 500` — 给 CMP 横幅 500ms 异步加载时间
- 全量重新生成 159 页，推送到两个仓库
