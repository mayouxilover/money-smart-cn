## 🚀 本地运行

### 前置要求

- Ruby 版本 >= 2.7（推荐 3.1+）
- Bundler（`gem install bundler`）

### 步骤

```bash
# 克隆仓库
git clone https://github.com/yourusername/money-smart-cn.git
cd money-smart-cn

# 安装依赖
bundle install

# 本地运行（默认 http://localhost:4000）
bundle exec jekyll serve --livereload
```
