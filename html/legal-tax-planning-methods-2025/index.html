7项专项附加扣除详解：每项能省多少税？（附计算案例）

扣除一：子女教育（每年12,000元/人）

条件： 子女年满3岁至博士毕业期间，均可扣除。

能省多少税？ 假设你适用10%税率，扣除12,000元 = 少交1,200元个税。

注意： 可以父母各扣50%，也可以一方扣100%。建议让收入高的一方全额扣除（省税更多）。

扣除二：3岁以下婴幼儿照护（每年12,000元/人）

条件： 2022年1月1日后出生的婴幼儿，0-3岁期间可扣除。

与子女教育的关系： 可以同时享受！0-3岁扣"婴幼儿照护"，3岁后改为"子女教育"，无缝衔接。

扣除三：住房贷款利息 vs 住房租金（二选一）

房贷利息扣除： 每年12,000元，需为首套住房贷款。

住房租金扣除： - 直辖市/省会/计划单列市：每年18,000元（每月1,500元） - 人口超100万城市：每年15,600元（每月1,300元） - 其他城市：每年12,000元（每月1,000元）

策略： 如果你在北上广深租房，选"租金扣除"（18,000元）比"房贷扣除"（12,000元）更划算，每年多省600元个税。

扣除四：赡养老人（每年最高36,000元）

2025年最大红利！ 扣除标准翻倍了。

家庭情况 每年扣除额 省税金额（按10%税率）
独生子女 ¥36,000 ¥3,600

操作技巧： 非独生子女需要兄弟姐妹协商分摊，并签署分摊协议（APP上直接操作即可）。建议让收入最高的兄弟姐妹多分摊，省税最大化。

扣除五：继续教育

  • 学历继续教育（如在职研究生）：每年4,800元，在读期间每年可扣
  • 职业资格继续教育（如CPA、律师资格、软考等）：取证当年一次性扣3,600元

提示： 职业资格扣除需要在《国家职业资格目录》内的证书才能扣。

扣除六：大病医疗

条件： 年度内医保系统记录的自付部分（即医保报销后自己付的钱）超过15,000元的部分，可以扣除，最高80,000元。

操作： 在"个人所得税"APP里，系统会自动统计你的医保自付金额，超过15,000元的部分直接点击确认即可。

扣除七：个人养老金账户抵税（2025年最值得用！）

这是2025年最强大的合法节税工具，很多人还不知道。

政策： 每年向个人养老金账户存入最高12,000元，可以在个税汇算清缴时全额扣除。

这5种情况可以合法退税，很多人不知道！

情况一：年中离职/换工作，预缴税款过多

如果你一年内换过工作，两家公司分别扣税，可能导致全年预缴税款超过实际应缴。汇算清缴时可以申请退税。

实例： 小王上半年月薪8,000元（不缴税），下半年月薪15,000元。全年收入13.8万，实际应缴个税约1,080元。但因为下家单位按月预扣，可能预扣了2,000多元，汇算清缴可退1,000元左右。

情况二：有公益慈善捐赠

条件： 通过正规公益组织（有捐赠票据）的捐赠，可以在应纳税所得额30%以内扣除。

实例： 年薪30万，捐了2万元公益款 → 应纳税所得额减少2万元 → 按20%税率算，省税4,000元。

情况三：有商业健康保险（税优健康险）

部分商业健康保险产品（保单上会标注"税优识别码"）可以享受每年2,400元的税前扣除。

注意： 普通商业保险（如重疾险、医疗险）不能税前扣除。只有带"税优识别码"的才，通常这类产品是税优健康险或税延养老险。

情况四：自由职业者有专项扣除没填报

自由职业者（自媒体、网约车司机、外卖骑手等）往往不知道自己也可以享受专项附加扣除。在做个税汇算清缴时，点击"专项附加扣除填报"，填好之后，往往能退一大笔钱。

情况五：年终奖单独计税 vs 合并计税，选错了

2025年，年终奖可以单独计税（不并入综合所得），也可以合并计税。

策略： 在个税APP汇算清缴时，系统会自动帮你比较两种方式，选缴税少的那种。很多人没注意这个选择，白白多交税。

自由职业者/兼职收入怎么合法节税？

自由职业者的收入，按劳务报酬所得计税，预扣税率20%起，很高！

合法节税方法

方法一：注册个体工商户，享受小微企业税收优惠

如果你是持续性的自由职业（如持续接设计单、持续做自媒体），可以考虑注册个体工商户。

2025年小微企业增值税起征点提高到月销售额10万元（季度30万元）以下免征增值税。

方法二：将部分收入转化为"经营所得"

劳务报酬所得税率20%-40%，而经营所得个税税率是5%-35%，且有很多扣除项。通过个体工商户申报，往往能降低税负。

⚠️ 注意： 此方法需要真实经营活动支撑，不能虚构业务开票，虚开发票是犯罪行为。

方法三：在税收洼地注册个体户（慎重！）

部分地区（如海南自贸港、西部大开发地区）有个税优惠政策。但2025年税务稽查越来越严，"异地虚开"风险极高，不建议普通自由职业者操作。

个人养老金账户抵税：每年最多能省多少钱？

这是2025年最值得操作的合法节税方式，零风险、100%合规。

操作步骤

  1. 在银行开立个人养老金账户（各大银行APP均可办理）
  2. 向账户存入资金（每年最高12,000元）
  3. 用账户资金购买符合规定的理财产品/基金/存款
  4. 次年3-6月个税汇算清缴时，扫码（或输入）个人养老金缴费凭证上的二维码/编码
  5. 系统自动扣除，退税到账

税收筹划的红线：这些"节税"方法千万别碰！

红线一：虚开发票

后果： 虚开增值税专用发票，金额超过5万元即可追究刑事责任（有期徒刑）。不要为了省几万税费，赌上人身自由。

红线二："阴阳合同"

典型案例： 明星签订"阴阳合同"偷漏税，已被查处多人（范冰冰、郑爽等）。

后果： 补缴税款+滞纳金+5倍以下罚款，金额巨大的追究刑事责任。

红线三：用他人身份证拆分收入

把本该属于你的收入，拆分成多个人的收入来避税。

后果： 属于偷税漏税，被查到后补缴+罚款，严重的追究刑事责任。

红线四：违规享受"税收洼地"政策

在没有任何实际经营的情况下，在税收洼地注册公司/个体户，只为了开发票。

后果： 2025年税务大数据已经非常强大，"异地虚开"被查概率极高。

年薪10万/20万/50万分别能省多少税？实例计算

实例一：年薪10万（税率3%-10%）

未填报专项扣除： 应纳税所得额 = 100,000 - 60,000（免税额）= 40,000元 → 个税 = 40,000×10% - 2,520 = 1,480元

填报后（子女教育12,000 + 房贷12,000 + 赡养老人18,000）： 应纳税所得额 = 40,000 - 42,000 = 0元 → 个税 = 0元

省税：1,480元/年

实例二：年薪20万（税率10%-20%）

未填报： 应纳税所得额 = 200,000 - 60,000 = 140,000元 → 个税 = 140,000×20% - 16,920 = 11,080元

填报后（各项扣除合计6万元）： 应纳税所得额 = 140,000 - 60,000 = 80,000元 → 个税 = 80,000×10% - 2,520 = 5,480元

省税：5,600元/年

实例三：年薪50万（税率20%-30%）

未填报： 应纳税所得额 = 500,000 - 60,000 = 440,000元 → 个税 = 440,000×25% - 31,920 = 78,080元

填报后（各项扣除合计6万元 + 个人养老金1.2万元）： 应纳税所得额 = 440,000 - 72,000 = 368,000元 → 个税 = 368,000×25% - 31,920 = 60,080元

省税：18,000元/年（其中个人养老金抵税贡献2,400元）

常见问题 FAQ

Q：个税专项扣除在哪里填报？

A：下载"个人所得税"APP（国家税务总局官方APP），注册登录后，首页点击"专项附加扣除填报"，选择扣除年度（如2025），逐项填写即可。填报后，单位发工资时会自动预扣，你每月到手工资就会多一点。

Q：个人养老金账户抵税，钱取出来时要补税吗？

A：不用补税。 个人养老金账户的钱，退休后取出时，不并入综合所得，而是单独按3%税率计税。如果你存入时税率是20%-45%，取出时只按3%缴税，净赚！

Q：自由职业者怎么申报个税和扣除？

A：自由职业者需要在每年3月1日-6月30日做个税汇算清缴。下载"个人所得税"APP，点击"综合所得年度汇算"，系统会引导你填报收入、专项扣除，最后算出应补/应退税额。很多人第一次汇算能退几千元，一定要做！

本文政策依据为国家税务总局2024-2025年发布的个税专项附加扣除政策文件，政策可能调整，请以最新官方文件为准。

税收筹划应在法律框架内进行，任何违法违规的"节税"行为都将承担法律责任。
