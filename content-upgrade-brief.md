### 共性问题（5篇全中）

#### A. 信息密度严重不足
- **债券投资入门**（700字）：概念解释占了60%，真正有价值的内容（策略对比、购买渠道、收益率数据）各写一句话带过
- **退休规划**（700字）：全文只有2个数字计算，都是最基础的复利运算，完全没有养老金替代率的分年龄段数据、不同城市生活成本差异等"高级信息"
- **仅基金定投一篇有表格和数据支撑**，其他4篇全是纯文字段落

#### B. "真实案例"有名无实
- 万能险文章声称"附3个真实投保案例"，实际每个案例压缩成一句话：`"张女士年交2万，5年后IRR约2.1%"`
- 缺少：投保时间、具体产品名称（可脱敏）、保费构成明细、退保现金价值走向图、与替代方案的真实对比
- **案例没有"人"的感觉**——没有年龄、职业、家庭情况、决策犹豫点

#### C. 缺少数据来源和权威引用
- 5篇文章中，**仅基金定投一篇提到了数据来源**（Wind资讯、天天基金、天相投顾）
- 其余4篇的"2025年结算利率普遍3.0%-3.3%""2024年分红实现率国寿约90%-110%"——完全无出处
- 读者无法判断这些数据是"作者编的"还是"真实统计的"

#### D. FAQ 像装饰品
- 每篇文章固定3个FAQ，回答长度1-2句话
- 例如债券FAQ："债券会亏本吗？持有到期不会亏本。但在到期前卖出可能下跌。"——这是百科级别的回答，不是"干货"
- 没有分层：新手常见问题 vs 进阶常见问题 vs 争议性问题

#### E. 结构千篇一律
- 所有文章都是：摘要 → 概念解释 → 要点罗列 → FAQ → 免责声明
- 缺少：**实操清单**（买了之后第一步做什么）、**决策流程图**、**对比评分表**、**工具/计算器推荐**
- 没有"这篇文章我能马上用"的感觉

#### F. 语气偏教科书，缺乏自媒体亲和力
- 典型句式："三差（死差、利差、费差）"——专业术语直接丢，不解释
- 缺少：口语化过渡（"很多人问我..."）、真实读者问题引入（"上周有个粉丝留言说..."）、场景化开篇
- 读起来像**保险从业资格考试教材**而非**帮普通人做决策的内容**

## 三、期望的内容升级标准

### 3.1 信息密度标准（每篇文章底线）

| 要素 | 最低要求 | 优秀标准 |
|------|---------|---------|
| 数据表格 | ≥1 个 | ≥3 个（含对比维度） |
| 真实案例 | ≥1 个完整案例（200字以上） | ≥2 个不同情境案例 |
| 数据出处 | 每个关键数字标注来源 | 附查询链接/查询方法 |
| 实操步骤 | ≥3 步可执行清单 | 附检查表/模板 |
| 工具/计算器推荐 | ≥0（不强制） | 推荐具体工具名称+使用方法 |
| 底层逻辑解释 | 至少1处"为什么"的深度解释 | 用通俗类比讲清楚机制 |

### 3.2 结构多样性要求

**禁止**所有文章使用同一模板。推荐结构类型：

| 文章类型 | 推荐结构 | 适用主题 |
|---------|---------|---------|
| 教程型 | 场景引入 → 原理拆解 → 分步操作 → 常见错误 → 检查清单 | 定投、开户、理赔 |
| 对比评测型 | 参数表 → 各维度打分 → 适合人群矩阵 → 结论推荐 | 保险产品对比、理财渠道对比 |
| 案例分析型 | 人物背景 → 决策过程 → 结果复盘 → 教训提炼 → "如果是你会怎么做" | 理赔、投资失误 |
| 深度报告型 | 数据总览 → 趋势分析 → 原因拆解 → 预测展望 → 行动建议 | 利率走势、行业分析 |
| 清单/工具型 | 一句话说明 → 清单正文 → 使用说明 → 更新机制 | 投保前检查清单、年度理财体检 |

### 3.3 语言风格要求

- **去教科书化**：避免"众所周知""所谓""即"等学术腔
- **加"人味"**：允许出现"我""你""我们"，允许使用"说实话""说人话就是""这笔账我给你算清楚"
- **保留专业深度**：不牺牲准确性，但用类比降低理解门槛（如"增额终身寿的现金价值就像一棵慢慢长大的树"）
- **每篇文章至少1个"金句"**——可截图传播级别的总结句

### 3.4 SEO与用户价值平衡

当前文章有明显的"堆关键词"痕迹（tags里塞了5个长尾关键词），但正文本身不扎实。升级原则：
- **正文信息密度优先**，SEO 优化为辅助
- 标题保留搜索友好性，但不做标题党
- 每个 FAQ 必须给出真正有用的深度回答（≥150字），而非 SEO 填充
